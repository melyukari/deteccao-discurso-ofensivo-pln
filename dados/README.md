# Bases de dados utilizadas

Este projeto utiliza três bases públicas:

## OLID (Offensive Language Identification Dataset)
https://www.kaggle.com/datasets/feyzazkefe/olid-dataset?select=olid-training-v1.0.tsv

## Toxic Comment Classification Challenge
https://www.kaggle.com/competitions/jigsaw-toxic-comment-classification-challenge/data?select=train.csv.zip

## Davidson Hate Speech Dataset
https://github.com/t-davidson/hate-speech-and-offensive-language/blob/master/data/labeled_data.csv
